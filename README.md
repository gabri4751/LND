El repositorio contiene la estructura completa del sitio web organizada para mantener el código claro y fácil de mantener. En la carpeta CSSs se encuentran todos los archivos de estilos separados por secciones de la página para mejorar la organización: generico.css; contiene los estilos generales reutilizados en todo el sitio, header.css y footer.css gestionan el diseño del encabezado y el pie de página comunes a todas las páginas, mientras que index.css, cursos.css, tienda.css, contacto.css y sobre-nosotros.css contienen los estilos específicos de cada página. En la carpeta htmls se encuentran las diferentes páginas del sitio (index.html, cursos.html, tienda.html, contacto.html y sobrenosotros.html), desde las cuales el usuario puede navegar entre las distintas secciones mediante el menú principal del header presente en todas ellas. La carpeta img almacena todos los recursos gráficos del proyecto organizados por categorías (Articulos, Carrousel, Cursos, Logos, RedesSociales y Tienda) para facilitar su localización y reutilización dentro del proyecto. El sitio incluye distintos efectos visuales implementados con CSS, como efectos hover en la mayoría de botones, enlaces y elementos interactivos para mejorar la experiencia del usuario, así como transiciones suaves que aportan dinamismo. La navegación está diseñada para ser intuitiva, permitiendo acceder fácilmente a cada sección desde el menú superior, mientras que la separación de estilos por archivos permite modificar cada página o componente sin afectar al resto del sitio, facilitando así el mantenimiento y la escalabilidad del proyecto.
